# Microsoft营销活动和广告组设置中的负面关键字字段的注释

>[!NOTE]
>
>* 每个关键字的最大长度为100个字符。
>* 您可以在营销活动和广告组设置中或从以下位置设置负面关键字： [!UICONTROL Keywords] > [!UICONTROL Negatives] 视图。 营销活动级别的负值适用于广告组级别。
>* 更改 [!DNL Microsoft Advertising] 关键字删除现有关键字，并使用新关键字ID创建新关键字。 但是，更改匹配类型不会删除现有关键字。
