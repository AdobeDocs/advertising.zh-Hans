# 上传Microsoft多媒体广告和响应式广告图像的说明

<!-- Only part of the "Image" field description -->

要上传图像，请执行以下操作：

1. 对于每个图像：
   1. 单击&#x200B;**[!UICONTROL +]**&#x200B;并从设备或网络中选择图像。
   1. 选择纵横比。
   1. 根据需要拖动并放置裁切框以选择图像的可查看部分，然后单击&#x200B;**[!UICONTROL Proceed]**。
1. 指定完图像后，单击&#x200B;**[!UICONTROL Upload]**。
