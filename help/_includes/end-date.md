# 某些营销活动设置和MS广告组设置中的“结束日期”字段

**[!UICONTROL End date]：**&#x200B;可能发出出价的最后日期。 默认值为&#x200B;*无结束日期*。 要在指定日期之前投标，请以MM/DD/YYYY格式输入日期，或单击![日历](/help/search-social-commerce/assets/calendar.png)并选择日期。
