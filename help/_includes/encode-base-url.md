# 编码帐户和营销活动设置中的基本URL字段

**[!UICONTROL Encode Base URL]：** （仅具有目标URL和跟踪类型[!UICONTROL EF Redirect]的帐户）最终用户浏览器地址栏中的URL是否包含字符编码（如`%3D`，而不是`=`）：

* *[!UICONTROL On]* （默认）：显示URL中的字符编码。

* *[!UICONTROL Off]：*&#x200B;显示未编码的URL。
