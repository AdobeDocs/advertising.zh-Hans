# 在帐户和促销活动设置中对基本URL字段进行编码

**[!UICONTROL Encode Base URL]：** (具有目标URL和跟踪类型的帐户 [!UICONTROL EF Redirect] 仅限)最终用户是否将看到字符编码(例如 `%3D` 而不是 `=`)，网址为：

* *[!UICONTROL On]* （默认）：显示URL中的字符编码。

* *[!UICONTROL Off]：* 显示未编码的URL。
