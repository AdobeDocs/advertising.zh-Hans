# 百度、GGL、MS、YJP的营销活动和广告组设置中的负关键字字段

**[!UICONTROL Negative Keywords]：** （可选）查询时不会触发广告的任何关键词。 使用以下语法，不带减号(`-`)：

* 负值广泛匹配： `keyword` (不支持 [!DNL Microsoft Advertising])
* 负片语匹配： `"keyword"`
* 负精确匹配： `[keyword]`

用逗号分隔多个值，或在单独行中输入它们。 在一个操作中，最多可以输入或粘贴2000个负关键字。
