# 大多数关键词和站点链接设置中的基本URL字段

>[!NOTE]
>
>* 基本URL可能包含登陆页面域或子域内的重定向，但登陆页面域外无重定向。 Google Ads从此URL中提取域，并为广告添加任何可选的显示路径以创建显示URL。
