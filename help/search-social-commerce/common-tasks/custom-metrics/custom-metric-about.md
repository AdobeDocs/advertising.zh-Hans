# 关于自定义量度

您可以配置自定义（派生）量度，这些量度是通过为广告商跟踪的标准流量量度和现有交易属性（收入量度）来计算的。 自定义量度可以很简单(例如 `[Cost]/[Registrations]`)或复杂程度不限。

您可以将自定义量度作为列包含在项目组合和促销活动管理视图以及报表中。

>[!NOTE]
>
>Adobe Analytics计算指标和高级计算指标不可用于您的计算。
