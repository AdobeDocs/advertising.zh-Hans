# 对列重新排序

1. （可用时）单击列标题右侧的![向下箭头](/help/search-social-commerce/assets/arrow-down-expand.png "向下箭头")。

1. 选择排序选项：

   * *升序排序*（A到Z，或1到10）

   * *降序排序*（Z到A，或10到1）
