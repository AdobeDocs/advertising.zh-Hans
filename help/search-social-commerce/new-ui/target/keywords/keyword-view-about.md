# （新UI）关于[!UICONTROL Keywords]视图

*Beta功能*

[!UICONTROL Manage] > [!UICONTROL Keywords]视图包含两个选项卡：

* [!UICONTROL Keywords]选项卡列出了所选广告商帐户的筛选视图中的所有关键字。

* [!UICONTROL Negatives]选项卡列出了所选广告商帐户的筛选视图中的所有负关键字。
