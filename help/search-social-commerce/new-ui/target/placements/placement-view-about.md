# （新UI）关于[!UICONTROL Placements]视图

*Beta功能*

[!UICONTROL Manage] > [!UICONTROL Placements]视图包含两个选项卡：

* [!UICONTROL Placements]选项卡列出了所选广告商帐户的筛选视图中的所有版面。

* [!UICONTROL Negatives]选项卡列出了所选广告商帐户的筛选视图中的所有负面版面。
