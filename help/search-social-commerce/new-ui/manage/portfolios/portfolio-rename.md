# （新UI）重命名项目组合

*Beta功能*

快速重命名项目组合，而无需打开完整的项目组合设置。

1. 在主菜单中，单击&#x200B;**[!UICONTROL Manage]>[!UICONTROL Portfolios]**。

1. 将光标悬停在项目组合行上并单击&#x200B;**[!UICONTROL ...]>[!UICONTROL Rename]**。

1. 编辑名称，然后单击&#x200B;**[!UICONTROL Apply]**。
