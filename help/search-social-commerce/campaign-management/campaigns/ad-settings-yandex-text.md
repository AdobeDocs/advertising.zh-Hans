# [!DNL Yandex] 文本广告设置

## [!UICONTROL Text Ad]

**[!UICONTROL Ad Title]：** 横幅（广告）的标题。 最大长度为33个字符，单个单词不能包含超过23个字符。

>[!NOTE]
>
>更改的广告文案 [!DNL Yandex] 广告会删除现有广告并创建一个具有相同属性的新广告。

**[!UICONTROL Description Line 1]：** 横幅（广告）的正文。 最大长度为75个字符，单个单词不能超过22个字符。

>[!NOTE]
>
>更改的广告文案 [!DNL Yandex] 广告会删除现有广告并创建一个具有相同属性的新广告。
