# 关于模型精度报告

模型准确性报表指明用于优化项目组合竞价、活动预算和竞价策略目标的成本和收入模型的准确性，这些模型适用于优化类型和活动竞价策略。

不准确性可能表明性能问题，因此您可能需要调整项目组合设置。 有关对预测不准确性进行故障排除的更多信息，请参阅“优化指南”的“项目组合”部分中的“关于项目组合的常见问题解答”，该指南可在Search、Social和Commerce中找到。<!-- verify convention for referencing Optimization Guide here -->
