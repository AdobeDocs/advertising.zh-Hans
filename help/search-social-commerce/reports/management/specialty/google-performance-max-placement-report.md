# [!UICONTROL Google Performance Max Placement Report]

*仅适用于[!DNL Google Ads]帐户，且仅具有最佳营销活动*

[!UICONTROL Google Performance Max Placement Report]包含在指定帐户中表现最佳的[!DNL Google Ads]营销活动中的投放位置的展示数据。 此报表包含目标URL所包含的每个投放位置的展示次数。 默认情况下，对于在指定数据范围内至少收到一个印象的每个投放位置和目标URL组合，数据包括一行，即使投放位置在报告期间被删除。 默认情况下，行按促销活动以升序排列。

使用此报告可以：

* 识别哪些网站、应用程序和[!DNL YouTube]渠道为你的广告提供服务。

* 识别和排除不相关或性能不佳的投放位置，以提高营销活动ROI。

* 通过构建强大的排除列表，确保您的广告仅显示在品牌安全的环境中。
