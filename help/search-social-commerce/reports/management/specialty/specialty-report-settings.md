|  | [!UICONTROL Data Aggregation] | ([!UICONTROL Forecast Accuracy Report only])应在报表的每一行上表示的时间单位： <i>[!UICONTROL Summary (the default)]</i>、<i>[!UICONTROL Daily]</i>、<i>[!UICONTROL Weekly]</i> （然后指定一周的第一天）、<i>[!UICONTROL Monthly]</i>、<i>[!UICONTROL Quarterly]</i>、<i>[!UICONTROL Yearly]</i>或<i>[!UICONTROL Day of Week]</i> （仅限基本报表）。<br><br>对于指定日期范围内的每个时间单位，报表将为每个关键字或其他项显示一行。 例如，如果您为关键词报告选择<i>[!UICONTROL Summary]</i>并且日期范围为<i>[!UICONTROL Last 7 Days]</i>，则该报告将针对每个关键词都包含一行，每个行汇总了七天内的总计和平均值。 带有<i>[!UICONTROL Daily]</i>数据聚合的同一报告在过去七天中的每一天都为每条关键字包含一行（每条关键字包含七行）。<br><br><b>注释：</b><ul><li>每小时使用&quot;[!UICONTROL Hourly]&quot;或&quot;[!UICONTROL Day of Week (Hourly)]&quot;聚合聚合数据时：</li><ul><li>您最多可以查看过去14天的数据。 该报告按24小时制显示数据。 “[!UICONTROL 0]”在广告商的时区包括00:01-00:59 (12:01-12:59 a.m.)和“[!UICONTROL 23]”包括23:01-23:59 (11:01-11:59 p.m.)。</li><li>不支持&quot;[!UICONTROL Device]&quot;列。</li><li>对于[!UICONTROL Ad Group Report]，不支持&quot;[!UICONTROL Data Based On]&quot;选项&quot;[!UICONTROL Portfolio Configuration During the Specified Dates]&quot;。</li><li>对于[!UICONTROL Ad Group Report]，您可以仅基于当前项目组合配置（使用“[!UICONTROL Data Based On]”选项“[!UICONTROL Current Portfolio Configuration]”），而不是基于指定日期的项目组合配置。</li></ul></ul> |
|  | [!UICONTROL Conversions Based on] | （仅限[!UICONTROL AdWords Shopping Performance Report]、[!UICONTROL Campaign Daily Impression Share Report]和[!UICONTROL Keyword Daily Impression Share Report]）如何报告转化数据：<ul><li><i>[!UICONTROL Transaction date]</i>（默认）：查看交易日期在指定时间段内发生的交易。 此选项显示指定时间段内已获得的收入。</li><li><i>[!UICONTROL Click date]：</i>查看在指定时间段内发生的点击导致的事务。 当项目组合在点击和交易之间存在重大延迟时，此选项可用于计算项目组合的每次点击历史收入，它指示一段时间内预期的收入行为。</li></ul> |
|  | [!UICONTROL Date Range] | 要为其生成数据的日期范围：<ul><li><i>[预设范围]：</i>常用时间增量列表，范围从<i>[!UICONTROL Today]</i>到<i>[!UICONTROL Last 180 Days]</i>。 默认值为<i>[!UICONTROL Last 7 Days]</i>，它报告有可用数据的最近七天。 <b>注释：</b> <i>[!UICONTROL Last Month]</i>、<i>[!UICONTROL Last 3 Months]</i>和<i>[!UICONTROL Last 6 Months]</i>显示前一个日历月份的数据。</li><li><i>[!UICONTROL Custom Date Range]：</i>指定开始日期和结束日期。 以YYYY/MM/DD或M/D/YYYY格式输入日期，或单击字段旁边的![日历](/help/search-social-commerce/assets/calendar.png "日历")并选择日期。</li></ul> |
