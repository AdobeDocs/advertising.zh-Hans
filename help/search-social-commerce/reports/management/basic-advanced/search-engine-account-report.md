# [!UICONTROL Search Engine Account Report]

此 [!UICONTROL Search Engine Account Report] 包含具有促销活动的一个或多个广告网络帐户的成本、点击次数和（可选）转化数据。 默认情况下，指定日期范围内每个时间单位的每个适用帐户包含一行，这些行首先按广告网络，然后按帐户，以升序排列。

您可以查看前36个月的数据。
