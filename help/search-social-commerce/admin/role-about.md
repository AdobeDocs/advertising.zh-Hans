# 关于角色

角色决定用户类型可以执行的权限。 每个用户都分配有一个角色，可以执行该角色允许的所有活动。

仅限具有以下特征的用户： [!DNL Adobe] 帐户管理员和管理员角色可以为用户分配角色。
