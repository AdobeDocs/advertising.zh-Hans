# AMO ID (s_kwcid)跟踪参数

*仅集成Adobe Advertising-Adobe Analytics的广告商*

Adobe Advertising使用AMO ID附加参数（也称为`s_kwcid`参数）与Adobe Analytics共享有关您的促销活动的数据，该参数由广告渠道和广告网络特定的元素组成。

有关如何将AMO ID参数添加到您的跟踪URL以及DSP和Search、Social和Commerce的AMO ID格式的信息，请参阅“[由 [!DNL Analytics]](/help/integrations/analytics/ids.md#amo-id)使用的Adobe Advertising ID”。
