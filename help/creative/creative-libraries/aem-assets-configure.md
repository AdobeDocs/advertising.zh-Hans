# 配置[!DNL Creative]以访问Adobe Experience Manager图像资源

*已关闭的测试版*

在将现有Experience Manager图像资产上传到[!UICONTROL Creative Libraries]之前，关联DSP帐户的广告商帐户设置必须指定Experience Manager帐户的组织ID。 请联系您的Adobe客户团队以配置此设置。
