# 创建自定义报表

1. 在主菜单中，单击&#x200B;**[!UICONTROL Reports]**。
1. 单击&#x200B;**[!UICONTROL New Custom Report]**。
1. 指定[报表设置](/help/dsp/reports/report-settings.md)。
1. 单击&#x200B;**[!UICONTROL Create Custom Report]**。
