# 运行自定义报表

>[!NOTE]
>
>您还可以在 [创建](report-create.md) 或 [编辑](report-edit.md) 它。

1. 在主菜单中，单击 **[!UICONTROL Reports]**.

1. 在 [!UICONTROL Options] 列，单击 **[!UICONTROL options]** > **[!UICONTROL Run Now]**.

1. 在确认消息中，单击 **[!UICONTROL OK]**.

   完成报表后，该报表会发送到报表设置中指定的所有电子邮件地址。
