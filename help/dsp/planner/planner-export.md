# 导出已连接电视覆盖计划的预测

*Beta版功能*

您可以导出 [!UICONTROL Budget vs Reach] 预测到 [!DNL Microsoft® Excel] 电子表格**带 [!UICONTROL Budget] 和 [!UICONTROL Reach] 列。

* 在生成的预测中，单击 **[!UICONTROL ...]** > **[!UICONTROL Export]** 在右上角。
