# 关于警报订阅

您可以创建称为订阅的自动电子邮件警报组，这些警报会在满足指定的帐户、促销活动、广告和专用库存交易参数时发送电子邮件通知。

例如，当营销活动花费了指定的预算金额时，您会收到警报。 此外，当私有库存交易在过去N天内收到的成功率低于指定的成功率时，您还可以收到警报。

您可以在订阅中添加或删除警报，编辑接收订阅电子邮件通知的用户，禁用和重新启用警报订阅，以及删除警报订阅。

>[!TIP]
>
> 如果您未主动检查营销活动报表，请在启动营销活动后尽快设置电子邮件报表。

## 可用警报

* 帐户

   * 帐户余额

   * 帐户支出

   * 已创建新的营销活动

* 营销活动

   * 无营销活动支出

   * 营销活动支出

   * 营销活动支出占上限的百分比

   * 无营销活动展示次数

   * Campaign eCPM

* 广告

   * 广告拒绝

* 信息源（专用库存交易）

   * 信息源获胜率

   * 信息源竞价率

   * 无信息源支出

   * 无信息源展示

   * 无默认投放位置的保证馈送
